一套实用的自动化工具集合，包括管道保温计算、桌面文件整理、Foxmail邮件整理等功能。

## 📦 工具列表

### 🔧 管道保温计算工具
- **`pipe_insulation_calculator.py`** - 化工管道保温表面积计算程序
- **`start_pipe_calc.bat`** - Windows一键启动脚本
- **`pipe_calc_guide.md`** - 详细使用指南

### 🖥️ 桌面文件整理工具
- **`desktop_organizer.py`** - 智能桌面整理（按文件类型分类）
- **`file_type_organizer.py`** - 高级文件类型分类工具
- **`simple_desktop_cleaner.py`** - 简易快速整理
- **`start_desktop_clean.bat`** - Windows启动脚本

### 📧 Foxmail邮件整理工具
- **`foxmail_ali_organizer.py`** - 阿里企业邮箱专用整理工具
- **`foxmail_300_quick.bat`** - 300封邮件快速整理脚本
- **`foxmail_organizer.py`** - 通用Foxmail整理策略生成器

### 📚 文档和指南
- **`docs/`** - 详细使用文档
- **`examples/`** - 使用示例
- **`config/`** - 配置文件示例
